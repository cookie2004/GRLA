VGBOT - General reinforcement learning agent for gameplay

File descriptions:

210414_SM_Explorer_Frame_stack.ipynb contains the RL agent class. Look at this file if you want to train your own agent.
210418_Model_eval.ipnyb contains the functions to generate a cool video showing individual layer output.

All the target and behavior files are the models trained on SMB. 210612_1_SMB_DQN_target is the best model. Use them or retrain at your leisure.
